This project was started to help win money via gambling on EPL matches. To run this code you will need to download the latest csv on the English Premier League season from https://www.football-data.co.uk/englandm.php . In the code, change the csv to whatever you want, but importantly, the column names must be changed in the csv in order to run, namely:
FTHG --> HomeGoals
FTAG --> AwayGoals
FTR --> Result
The csv gets updated fairly frequently so constantly check to make sure you have the latest version otherwise your results may be off.
